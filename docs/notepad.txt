## Ideas

### Device Simulator
Create a device simulator that uses real messages from database:
- devices (brands, types)
- messages (types, payloads)
- timing (when to send messages)
- enabled (on/off)

Use for testing responses, finding problems, flooding tests.

### RF Config Validation
Check on startup if ramses_cc has:
- Bound REM configured
- Enable Message Events with regex: 31DA|10D0
- Enable Send Packet Service
Log errors if not configured.
